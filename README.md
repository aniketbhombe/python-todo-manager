# Python Task Manager Desktop App

This is a desktop To-Do application built using Python and Tkinter.

## Features
- Add tasks
- Task priority (High / Medium / Low)
- Due date
- Mark task complete
- Search tasks
- Export tasks to CSV
- Persistent storage using JSON

## Technologies
- Python
- Tkinter
- JSON
- CSV

## How to Run

Install Python and run:

python todo_app.py
